Bienvenido a STEAD ("Sistema de Tutoría en Estadística y Análisis de Datos")

Descripcion del proyecto

Este proyecto tiene como finalidad desarrollar un prototipo de sitio web que represente un sistema básico para la gestión de tutorías en las áreas de estadística y análisis de datos, dirigido tanto a estudiantes como a docentes. Mediante este prototipo se busca mostrar, de forma sencilla, cómo podría organizarse el proceso de solicitud y acceso a tutorías académicas a través de una plataforma web.

Esta idea surge a partir de la necesidad de contar con un espacio digital que ayude a organizar mejor las tutorías, ya que en muchos casos este proceso se realiza de manera informal. Aunque el sistema no funcionará de forma automática, permitirá tener una visión general de cómo podría estructurarse una plataforma de este tipo, sirviendo como un primer acercamiento al diseño del sistema, en el cual tanto los estudiantes como docentes puedan suministrar recursos para el análisis de los datos.

Mision

  • Desarrollar un sistema web que permita la reserva y acceso a tutorías en estadística y análisis de datos.
  • Proporcionar un aprendizaje continuo y eficaz en el análisis de datos.
  • Facilitar la interacción entre estudiantes y docentes.

Vision

  1. Desarrollo de un sistema web interactivo.
  2. Implementación de una base de datos para gestionar reservas y tutorías.
  3. Integración con herramientas de análisis de datos.

Alcance del proyecto

El proyecto comprenderá el diseño y desarrollo de varias páginas web, mediante las cuales estarán enlazadas entre sí para relevar el funcionamiento del sistema. Estas páginas permitirán representar acciones básicas como el registro de usuarios, el inicio de sesión, la consulta de disponibilidad de docentes y la solicitud de tutorías.

Además, se incluirán formularios y tablas informativas con el objetivo de facilitar la comprensión del funcionamiento del sistema. Es importante señalar que el proyecto se presentará únicamente como un prototipo demostrativo, por lo que no se implementará base de datos.
